## 二、Android 项目创建步骤

### 2.1 在 Android Studio 中创建项目

1. **File → New → New Project**
2. **选择 "Empty Activity"**
3. **配置项目**：
   - Name: `TestWings`
   - Package name: `com.testwings`
   - **Save location: `E:\AutoTestDemo\TestWings\android`** ⭐ 重要！
   - Language: `Kotlin`
   - Minimum SDK: `API 29` (Android 10)
4. **点击 Finish**

### 2.2 验证项目结构

创建完成后，项目结构应该是：

```
TestWings/
├── 00-README.md
├── docs/
├── android/          # Android 项目在这里
│   ├── app/
│   ├── build.gradle.kts
│   └── ...
└── .gitignore
```

### 2.3 如果项目已创建在根目录

**迁移步骤：**

1. 关闭 Android Studio
2. 创建 `android/` 目录
3. 移动以下文件到 `android/`：
   - `app/`
   - `build.gradle.kts`
   - `settings.gradle.kts`
   - `gradle/`
4. 在 Android Studio 中打开 `android/` 目录

## 三、常见问题

### Q1: 如果 Save location 选择根目录会怎样？

**A:** Android Studio 会提示目录不为空，建议选择在子目录创建。

### Q2: 会覆盖现有文件吗？

**A:** 不会！Android Studio 只会创建新文件，不会覆盖现有文件（如 00-README.md、docs/ 等）。

### Q3: 文档和代码在同一个项目，会不会混乱？

**A:** 不会！使用子目录结构（`android/`），文档和代码分离清晰。

### Q4: 如果已经创建了项目，想移动怎么办？

**A:** 
1. 在 Android Studio 中关闭项目
2. 将 Android 项目文件夹移动到 `TestWings/android/`
3. 在 Android Studio 中重新打开项目（File → Open → 选择 `android/` 目录）
