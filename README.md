# TestWings / 测试之翼

> 给测试一双翅膀，让自由成为可能  
> Give tests wings, set engineers free

## 项目简介

TestWings 是一个借助 AI 能力协助软件测试工程师完成移动终端应用自动化测试的智能平台。我们的使命是解放测试工程师，让他们从重复繁琐的工作中解脱出来，追求真正自由的工作和人生。

## 核心理念

- **自由**：让测试工程师获得工作的自由，更获得人生的自由
- **智能**：借助 AI 能力，自动化处理测试任务
- **专注**：让工程师专注于更有价值的创造性工作

## 功能特性

- 🤖 **AI驱动的测试用例理解**：支持自然语言描述测试用例，AI自动理解并生成测试计划
- 👁️ **智能屏幕识别**：结合OCR、计算机视觉和Accessibility Service，准确识别UI元素
- 🎯 **智能元素定位**：基于语义匹配，自动定位目标元素，适应UI变化
- ⚡ **自动化执行**：像人一样操作手机，执行点击、输入、滑动等操作
- ✅ **智能结果验证**：自动验证测试结果，生成详细测试报告
- 🔄 **自适应学习**：从测试执行中学习，持续优化识别和执行准确率

## 技术原理

本项目灵感来源于：
- **豆包手机助手**：能够理解用户指令，自主完成手机上的复杂任务
- **象棋APP连线功能**：实时分析棋盘，提供最优走法建议

核心原理：**实时屏幕识别 + AI推理决策 + 自动化操作执行**

详细技术原理分析请参考：[技术原理分析文档](docs/技术原理分析.md)

## 架构设计

系统采用分层架构设计，包含：
- 用户接口层：Web管理端、移动端APP、CLI命令行
- 测试用例管理层：用例解析、存储、执行
- AI核心引擎层：用例理解、屏幕理解、操作规划
- 设备控制层：屏幕捕获、元素定位、操作执行
- 结果分析层：结果验证、报告生成、问题分析

详细架构设计请参考：[架构设计文档](docs/架构设计.md)

## 平台支持

### Android 开发
- **Android 10+**：完整支持
- **HarmonyOS 4.2.0+**：通过 Android 兼容层支持，可正常使用
- **开发工具**：Android Studio
- **开发语言**：Kotlin（推荐）或 Java

**开发策略**：使用 Android SDK 开发，兼容 HarmonyOS。后续可考虑 HarmonyOS 原生支持（类似 iOS）。

详细开发方案请参考：[Android 开发方案](docs/Android开发方案.md)

### 模型部署策略

**优先方案：完全本地部署**
- 所有 AI 模型部署在手机本地
- 完成完整的用例分析、执行、图像识别和结果输出
- 降低测试环境复杂度，无需搭建服务器
- 保护数据隐私，离线可用

**可选方案：企业级混合部署**
- 针对企业级大规模使用场景
- 可单独搭建 LLM 模型服务器
- 本地：OCR + UI 检测（轻量级模型）
- 云端：用例理解 + 屏幕理解 + 操作规划（大模型）

**硬件配置要求：**
- **最低配置**：8GB 内存（可使用轻量级 LLM）
- **推荐配置**：12GB+ 内存（可使用完整 LLM 功能）

详细部署方案请参考：[模型部署方案](docs/模型部署方案.md)

## 技术栈

### 移动端（Android）
- **开发语言**: Kotlin（推荐）或 Java
- **屏幕捕获**: MediaProjection API
- **UI访问**: AccessibilityService
- **操作执行**: AccessibilityService / UIAutomator2
- **数据库**: SQLite (Room)
- **异步处理**: Kotlin Coroutines

### AI模型（完全本地部署）
- **OCR**: PaddleOCR Mobile
- **UI检测**: YOLOv8 Nano
- **大语言模型**: Qwen-7B-Chat（INT8 量化）或 Qwen-1.8B-Chat（INT8 量化）
- **视觉-语言模型**: Qwen-VL-Chat（INT8 量化，可选）
- **推理框架**: ONNX Runtime, TensorFlow Lite

### 企业级服务（可选）
- **Web框架**: FastAPI
- **数据库**: PostgreSQL, Redis
- **LLM服务**: vLLM, TensorRT-LLM

### 开发环境准备

**重要：Android Studio 是必需的！**

Android Studio 是 Google 官方提供的专门用于 Android 开发的完整 IDE，包含：
- Android SDK 管理
- 项目创建和管理
- 代码编辑和调试
- 设备连接和运行
- APK 打包

**Cursor 无法替代 Android Studio**，因为 Android 开发需要完整的工具链。

**详细说明请参考**：[开发环境说明](docs/开发环境说明.md)

#### 快速开始步骤：

1. **安装 Android Studio**（必需）
   - 下载：https://developer.android.com/studio
   - 安装 Android SDK (API 29+)

2. **创建项目**
   - File → New → New Project
   - 选择 "Empty Activity"
   - 语言选择 "Kotlin"
   - 最低 SDK：API 29 (Android 10)

3. **连接设备**
   - 开启 USB 调试
   - 运行第一个 Hello World

**完成这些后，查看 [快速开始指南](docs/快速开始-第一个功能.md) 开始第一个功能！**
